# Linux SysAdmin Lab

Repositorio donde documento mi aprendizaje práctico de administración de sistemas Linux.

Objetivos:

- Administración de Ubuntu Server
- Administración de AlmaLinux
- Bash scripting
- Networking
- Systemd
- SSH
- Firewalld
- Git
- Automatización

Todo el laboratorio fue realizado sobre máquinas virtuales utilizando Hyper-V.
