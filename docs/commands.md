# Listado de comandos utilizados
- `tree`: permite la visualizaicón de archivos en una ruta específica en formato de árbol. Por defecto, si no se especifica la ruta, toma el directorio actual.
- `touch`: según `man`, cambia el timestamp de un archivo. Si el archivo no existe, lo crea. No permite crear directorios.
- `man`: manual de comandos.
- `mkdir`: permite crear directorios.
- `cd`: comando para moverse entre directorios.
- `ls`: lista archivos en una ruta específica. Por defecto, si no se especifica la ruta, toma el directorio actual. El parámetro `-l` imprime en patalla en formato de lista
  junto con información útil como permisos, tamaño y tipo de archivo, usuario owner.
- `mv`: mover archivo de ubicación. Se espcifica primero el archivo en cuestión y luego la ruta a la que se desea mover.
- `cp`: copiar archivo. Se espcifica primero el archivo en cuestión y luego la ruta a la que se desea mover.
